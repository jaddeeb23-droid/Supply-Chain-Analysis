# Supply Chain Analysis

Exploratory data analysis on a supply chain dataset (100 products across skincare, haircare, and cosmetics categories).

## Key Findings
- Skincare generates the highest revenue (~$241K), outperforming cosmetics by 33%
- Shipping costs are nearly identical across all carriers (~$5.50 avg), suggesting no cost advantage in carrier selection
- No correlation between lead time and shipping cost

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Dataset
[Supply Chain Analysis - Kaggle](https://www.kaggle.com/datasets/harshsingh2209/supply-chain-analysis)
